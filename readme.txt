🎸 리듬 게임 실행 가이드
이 프로젝트는 Processing(프로세싱) 환경에서 제작된 리듬 게임입니다. 다른 PC에서 실행하기 위해 아래의 단계를 따라주세요.

1. 필수 환경 설정
- Processing IDE 설치: processing.org에서 본인의 운영체제에 맞는 버전을 설치하세요.
- 라이브러리 설치: 이 코드는 외부 사운드 라이브러리를 사용합니다.
  Processing 실행 후 메뉴에서 [Sketch] -> [Import Library...] -> [Manage Libraries...] 선택
  검색창에 Minim 입력 후 설치 (작성자: ddf.minim)
  (Serial 라이브러리는 기본 내장되어 있으나, 혹시 없다면 동일한 방식으로 설치하세요.)

2. 아두이노 연결
- 아두이노를 컴퓨터와 연결합니다.
- 게임을 실행하여 아두이노가 정상작동하면 그대로 사용하면 됩니다.
- 만약에 아두이노가 실행이 안 된다면, 파일의 .ino파일을 업로드해 주세요 

@@주의@@ Processing과 아두이노가 동일한 Serial Port를 사용하기 때문에 두 프로그램이 동시에 켜져 있으면 한 쪽이 작동하지 않습니다. 꼭 다른 한 쪽을 종료 한 뒤 사용해 주세요

3. Processing 실행
- Processing의 왼쪽 위 플레이 버튼(▶)을 눌러서 프로그램을 실행해 주세요.
- 실행한 파일을 손님 모니터에 올린 뒤 전체화면 해주시면 됩니다
